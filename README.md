# 🔑 Modern Login UI Page

A sleek, responsive, and modern **Login Page UI** built using pure **HTML5** and **CSS3**. This project focuses on clean design, smooth user experience, and responsive layout across all device sizes.

---

## ✨ Features

- 📱 **Fully Responsive:** Looks great on mobile, tablet, and desktop screens.
- 🎨 **Modern Design:** Clean typography, balanced colors, and subtle UI effects (hover states/glassmorphism).
- ⚡ **Lightweight & Fast:** Pure HTML & CSS with zero external frameworks or heavy libraries.
- 🔍 **Form Styling:** Custom styled input fields, buttons, and "Remember Me / Forgot Password" options.
